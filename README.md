# MAX ↔ Telegram Bridge Bot

Двусторонний мост между мессенджером **MAX** и **Telegram**-группой.

---

## Что умеет

- **MAX → TG** — все сообщения из выбранного чата пересылаются в Telegram-группу: текст, фото, видео, документы, голосовые, стикеры.
- **TG → MAX** — участники Telegram пишут в MAX от своего имени после простой регистрации.
- **Регистрация** — командой `+имя Иван Иванов`. Имя можно менять в любой момент.
- **Нормализация фото** — очищает EXIF-метаданные перед отправкой (требует Pillow).
- **Braille-маркер** — скрытый маркер `[⠃медиа⠇]` предотвращает дублирование медиа при пересылке обратно.
- **Авто-фоллбек** — если фото не принимается как Photo, отправляется как Document.
- **Авто-реконнект** — при обрыве соединения с MAX клиент переподключается автоматически.
- **Загрузка истории** — при старте подгружает последние `HISTORY_LIMIT` сообщений из MAX.

---

## Требования

- Python 3.11+
- Аккаунт **MAX** с номером телефона
- **Telegram Bot Token** (от [@BotFather](https://t.me/BotFather))
- Bot должен быть добавлен в Telegram-группу с правами отправки сообщений

### Зависимости

```
aiogram
aiohttp
pymax
Pillow        # опционально, для нормализации фото
```

Установка:

```bash
pip install aiogram aiohttp pymax Pillow
```

---

## Настройка

Создайте файл `.env` (или задайте переменные окружения):

```env
MAX_PHONE=+79001234567
TG_BOT_TOKEN=123456789:AABBcc...
TG_GROUP_ID=-1001234567890
WORK_DIR=cache          # папка для временных файлов (по умолчанию cache)
HISTORY_LIMIT=200       # сколько сообщений загружать из истории при старте
```

> **TG_GROUP_ID** — ID группы, куда бот будет пересылать сообщения.  
> Узнать его можно, добавив [@userinfobot](https://t.me/userinfobot) в группу.

---

## Запуск

```bash
python bot.py
```

**Первый запуск** — бот выведет список чатов/диалогов MAX в консоль и попросит выбрать номер. Выбор сохраняется в `bridge_config.json`, повторный выбор не нужен.

---

## Структура файлов

```
bot.py               — основной скрипт
bridge_config.json   — сохранённый ID выбранного MAX-чата (создаётся автоматически)
bridge_db.json       — БД: маппинг сообщений и зарегистрированные пользователи
cache/               — временные медиафайлы (удаляются автоматически)
```

---

## Команды в Telegram

| Команда | Описание |
|---|---|
| `+имя Иван Иванов` | Зарегистрироваться / сменить имя |

После регистрации все сообщения из TG автоматически пересылаются в MAX от имени пользователя.

---

## Поддерживаемые типы медиа

| Тип | MAX → TG | TG → MAX |
|---|:---:|:---:|
| Текст | ✅ | ✅ |
| Фото | ✅ | ✅ |
| Документ | ✅ | ✅ |
| Голосовое | ✅ | ⚠️ уведомление |
| Видео | ✅ | ⚠️ уведомление |
| GIF / Стикер | ✅ | ⚠️ уведомление |

> ⚠️ — тип не поддерживается в TG→MAX, в MAX отправляется текстовое уведомление.
